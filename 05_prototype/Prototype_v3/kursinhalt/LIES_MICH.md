# Kursinhalt-Ordner

Hier kommen die vom Dozenten generierten JSON-Dateien hin:

- `kurs_manifest.json` — Kursinfo + Themenliste (Pflicht)
- `thema_1.json`, `thema_2.json`, ... — eine Datei pro Thema

Die App lädt diese Dateien automatisch, wenn sie via **Live Server** geöffnet wird
(VS Code → rechte untere Leiste → "Go Live").
